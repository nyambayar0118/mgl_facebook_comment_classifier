# QUICK START GUIDE
# Хурдан эхлэх заавар

## 1. Суулгах

```bash
# Шаардлагатай сангуудыг суулгах
pip install -r requirements.txt
```

## 2. Ажиллуулах

```bash
# Application ажиллуулах
python main.py
```

## 3. Алхам алхмаар заавар

### Жишээ 1: Naive Bayes ашиглаж сэтгэгдэл ангилах

```
1. Main menu дээр '1' дарж өгөгдөл ачаалах
   - Google Sheets URL сонгох (default)
   
2. '2' дарж Naive Bayes сургах
   - Alpha [1.0]: Enter дарах (default)
   - N-gram [2]: Enter дарах (unigram+bigram)
   - Test size [0.3]: Enter дарах
   
3. '4' дарж моделийг үнэлэх
   - Accuracy, Confusion Matrix харна
   
4. '5' дарж өөрийн сэтгэгдэл ангилуулах
   - Сэтгэгдэл: "Сайн байна уу, баярлалаа!"
   - Таамаглал харна (spam эсвэл ham)
   
5. '6' дарж график харах
   - '7' дарж бүх графикийг харах
```

### Жишээ 2: Decision Tree ашиглах

```
1. Main menu дээр '1' дарж өгөгдөл ачаалах
   
2. '3' дарж Decision Tree сургах
   - Max depth [8]: Enter дарах
   - Test size [0.2]: Enter дарах
   - Tree бүтэц харах эсэхийг сонгох
   
3. '4' дарж моделийг үнэлэх
   
4. '5' дарж өөрийн сэтгэгдэл ангилуулах
   - Шинжүүдийг оруулах (0 эсвэл 1)
```

## 4. Өөрийн өгөгдөл ашиглах

### CSV файл ашиглах:
```
1. Main menu дээр '1' дарах
2. '2' сонгох (CSV файл)
3. Файлын зам: fb_comment.csv
```

### XLSX файл ашиглах:
```
1. Main menu дээр '1' дарах
2. '3' сонгох (XLSX файл)
3. Файлын зам: fb_comment.xlsx
```

## 5. Санамж

- Эхлээд өгөгдөл ачаална (Сонголт 1)
- Дараа нь модель сургана (Сонголт 2 эсвэл 3)
- Моделийг үнэлж болно (Сонголт 4)
- Өөрийн сэтгэгдэл ангилуулж болно (Сонголт 5)
- График харж болно (Сонголт 6)

## 7. Жишээ таамаглалууд

### Naive Bayes-аар:
- "Сайн байна уу" → HAM
- "Click here to win money!" → SPAM
- "Баярлалаа, маш их таалагдлаа" → HAM
- "FREE iPhone!!! Click link!!!" → SPAM

### Decision Tree-ээр:
Дараах шинжүүдтэй:
- Link агуулсан: 1
- Email агуулсан: 1
- Монгол нэр: 0
→ SPAM магадлалтай
